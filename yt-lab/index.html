أكواد جاهزة (YouTube API + أدوات مساعدة)

JavaScript (متصفح): جلب الفيديوهات الأكثر رواجاً

// ملاحظة: يفضّل هذا الكود على الخادم لتجنب كشف المفتاح
async function fetchTrending(regionCode = "US") {
  const API_KEY = "YOUR_YOUTUBE_DATA_API_KEY";
  const url = new URL("https://www.googleapis.com/youtube/v3/videos");
  url.search = new URLSearchParams({
    part: "snippet,statistics,contentDetails",
    chart: "mostPopular",
    maxResults: "10",
    regionCode,
    key: API_KEY
  }).toString();

  const res = await fetch(url);
  if (!res.ok) throw new Error("YouTube API error");
  const data = await res.json();
  return data.items.map(v => ({
    id: v.id,
    title: v.snippet.title,
    channel: v.snippet.channelTitle,
    views: v.statistics.viewCount
  }));
}

fetchTrending("SA").then(console.log).catch(console.error);

Node.js: رفع فيديو (OAuth2)

// npm i googleapis
// أنشئ OAuth 2.0 Client من Google Cloud وحدد redirect URI
const { google } = require("googleapis");
const fs = require("fs");

async function uploadVideo({
  clientId, clientSecret, redirectUri, tokens,
  filePath, title, description, tags = []
}) {
  const oauth2Client = new google.auth.OAuth2(clientId, clientSecret, redirectUri);
  oauth2Client.setCredentials(tokens);

  const youtube = google.youtube({ version: "v3", auth: oauth2Client });
  const res = await youtube.videos.insert({
    part: ["snippet", "status"],
    requestBody: {
      snippet: { title, description, tags },
      status: { privacyStatus: "public" }
    },
    media: {
      body: fs.createReadStream(filePath)
    }
  });
  console.log("Uploaded:", res.data.id);
}

// الخطوة الأولى: احصل على كود الموافقة لإنشاء tokens ثم خزّنه بأمان.

Python: البحث في اليوتيوب

# pip install google-api-python-client
from googleapiclient.discovery import build

def yt_search(api_key: str, q: str, max_results: int = 10):
    yt = build("youtube", "v3", developerKey=api_key)
    req = yt.search().list(
        part="snippet",
        q=q,
        type="video",
        maxResults=max_results,
        order="viewCount"
    )
    res = req.execute()
    return [
        {
            "videoId": i["id"]["videoId"],
            "title": i["snippet"]["title"],
            "channel": i["snippet"]["channelTitle"]
        }
        for i in res.get("items", [])
    ]

if __name__ == "__main__":
    print(yt_search("YOUR_KEY", "تعلم بايثون"))

دليل نمو عملي لقناتك

صيغة فيديو فعّالة

  • الثواني 0-10: Hook قوي (نتيجة/وعد/سؤال مصيري).
  • تعزيز الفض��ل: لا تعطِ كل الإجابة مبكراً.
  • إيقاع سريع: كل 5-10 ثواني تغيير بصري/صوتي.
  • قيمة مركزة: 1-3 أفكار رئيسية، أمثلة عملية.
  • دعوة للاشتراك: بعد تقديم قيمة ملموسة.
  • نهاية: CTA واضح لفيديو مكمل.

تحسين CTR واحتفاظ المشاهد

  • عنوان قصير وواضح مع فضول محسوب.
  • مصغّرة متباينة الألوان ونص كبير من كلمتين-ثلاث.
  • أول 30 ثانية تحدد الاستمرار: تجنب المقدمات الطويلة.
  • فصول واضحة تسهل التنقل وتحسن الرضا.

إستراتيجية نشر

  • ثبات الجدول أهم من الكثرة المفاجئة.
  • اختبر توقيتات مختلفة ثم اثبت على الأفضل.
  • حوّل فيديو واحد إلى عدة مقاطع Shorts.

تحليل منافسين

  • اجمع أعلى 50 فيديو في تخصصك، استخرج صيغ العناوين المتكررة.
  • قارن أول 15 ثانية، واعرف ما يفصل القوي عن العادي.
  • جد فجوة: موضوعات مطلوبة لم تُغطَّ بعمق.

إعدادات (اختياري)

للتكاملات المتقدمة، نفّذ الاستدعاءات من خادمك لحماية المفاتيح. تخزين المفاتيح على المتصفح غير آمن.
